# D. 交互问题

题意：环上均匀分布mmm个点,初始有3个指针指向这些点，每次拨动某一个指针使其指向a[i]a[i]a[i]点，不能连续两次拨动同一个指针，求最小拨动总距离。（拨动距离为0也算拨动）

分析: 如果三个三个操作看的话，会发现第i+1i+1i+1次操作和第i−1i-1i−1次操作之间不会有“不能连续两次拨动同一个指针”的限制要求，处理起来会方便很多。且我们还可以得出一个结论：第aia_ia​i​​操作后必有一个指针是指向aia_ia​i​​的。

DP：设dp[i][j]dp[i][j]dp[i][j]为处理完aia_ia​i​​，三个指针分别指向aia_ia​i​​（这是必然的）、ai−1a_{i-1}a​i−1​​（不能连续拨动同一个所以会有指向ai−1a_{i-1}a​i−1​​的）、jjj。接着转移dpdpdp：

  • 第i+1i+1i+1个位置，可以拨动ai−1a_{i-1}a​i−1​​指针，即：dp[i+1][j]=min(dp[i][j]+dis[ai−1][ai+1],dp[i+1][j])dp[i+1][j]=min(dp[i][j]+dis[a_{i-1}][a_{i+1}],dp[i+1][j])dp[i+1][j]=min(dp[i][j]+dis[a​i−1​​][a​i+1​​],dp[i+1][j])
  • 也可以拨动指向j的指针，即：dp[i+1][ai−1]=min(dp[i][j]+dis[j][ai+1],dp[i+1][ai−1])dp[i+1][a_{i-1}]=min(dp[i][j]+dis[j][a_{i+1}],dp[i+1][a_{i-1}])dp[i+1][a​i−1​​]=min(dp[i][j]+dis[j][a​i+1​​],dp[i+1][a​i−1​​])

int DIS[5010][5010];
void solve() {
	sii(n,m);
	siii(s1, s2, s3);
	for (int i = 1; i <= n; i++)si(a[i]);
	for (int i = 1; i <= m; i++)
		for (int j = 1; j <= m; j++)
			DIS[i][j] = dis(i, j);
	memset(dp, 63, sizeof(dp));
	dp[2][s3] = min(DIS[s1][a[1]] + DIS[s2][a[2]], DIS[s2][a[1]] + DIS[s1][a[2]]);
	dp[2][s1] = min(DIS[s2][a[1]] + DIS[s3][a[2]], DIS[s3][a[1]] + DIS[s2][a[2]]);
	dp[2][s2] = min(DIS[s1][a[1]] + DIS[s3][a[2]], DIS[s3][a[1]] + DIS[s1][a[2]]);
	if (s1 == a[1]) dp[2][a[1]] = min(dp[2][a[1]],dp[2][s1]);
	if (s2 == a[1]) dp[2][a[1]] = min(dp[2][a[1]], dp[2][s2]);
	if (s3 == a[1])dp[2][a[1]] = min(dp[2][a[1]], dp[2][s3]);
	if (s1 == a[2])dp[2][a[2]] = min(dp[2][a[1]], dp[2][s1]);
	if (s2 == a[2])dp[2][a[1]] = min(dp[2][a[1]], dp[2][s2]);
	if (s3 == a[2])dp[2][a[1]] = min(dp[2][a[1]], dp[2][s3]);
	for (int i = 2; i <= n; i++) {
		for (int j = 1; j <= m; j++) {
			dp[i + 1][j] = min(dp[i][j] + DIS[a[i-1]][a[i+1]], dp[i + 1][j]);//指向a[i-1]的拨向a[i+1]
			dp[i + 1][a[i - 1]] = min(dp[i][j] + DIS[j][a[i+1]], dp[i + 1][a[i - 1]]);//指向j的拨向a[i+1]
		}
	}
	int ans = 1e9;
	for (int i = 1; i <= m; i++)
		ans = min(ans, dp[n][i]);
	printf("%d", ans);
}
